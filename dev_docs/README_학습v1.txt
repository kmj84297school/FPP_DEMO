FPP XGBoost v1 — 학습 기록 (2026.7)
====================================
코호트: 2018~2020 피처 시즌, 만 23세 이하, 900분+ 필드플레이어
       1,323 선수-시즌 (841명), 라벨 창 t+2~t+3 (검열 없음)
라벨:  ① survived — 2~3년 후 빅5에서 900분+ 시즌 존재 여부 (잔존율 75.1%)
       ② fut_ability — 잔존자의 미래 능력점수 (t+3 우선, 993행)
검증:  GroupKFold 5겹, 선수 단위 분할 (동일 선수 훈련/검증 동시 포함 차단)

결과:
- 잔존 분류 AUC 0.718 ± 0.011
- 미래 능력 회귀 MAE 6.73 / R² 0.169
- 기저모델 비교: "미래=현재" 대비 +2.3%, "전체 평균" 대비 +11.4% 개선
- 지배적 피처: 현재 능력점수(0.115) > 포지션 점수 > FW 여부 > 스타일 점수

정직한 해석:
현재 능력이 미래의 최강 예측자이며(상관 0.547), XGBoost의 순수 추가 기여는
아직 작다(+2.3%). 이는 실패가 아니라 잠재력 예측 문제의 난이도 실측이다.

v2 개선 레버 (우선순위):
1. 성장 궤적 피처 추가 — 전 시즌 대비 변화량(Δ스탯). 현재 모델은 단일 시즌
   스냅샷만 봐서 "성장 중인 선수"와 "정점인 선수"를 구분 못 함
2. 라벨 노이즈 축소 — 미래 단일 시즌 대신 t+2~t+4 평균 능력점수
3. 부트스트랩 신뢰구간 — 예측의 불확실성 정량화 (다음 단계 확정 작업)
4. 유사 선수 레이어 — k-NN 실증 분포와 교차 검증

v2 학습 기록 (같은 날)
====================
변경: ① 성장 궤적 Δ피처 13종 추가(전 시즌 600분+ 대비 변화량, 보유율 39.5%)
      ② 라벨 노이즈 축소 — 미래 단일 시즌 → t+2·t+3 자격 시즌 평균
결과: 회귀 기저 대비 개선 +2.3% → +5.0% (R² 0.231), 분류 AUC 0.712(동일 수준)
발견: Δ계열 중요도 합 25%. d_grp_progress(전진 능력 성장분)가 Δ 중 1위 —
      수준(level)보다 기울기(slope)가 미래 신호라는 자체 발견
한계: v1/v2 MAE 직접 비교 불가(라벨 상이) → 기저 대비 개선폭으로 비교함
      Δ 보유율 39.5% (2018 코호트는 전 시즌 데이터 부재)

부트스트랩 신뢰구간 (v2 모델 기반)
================================
방법: 선수 단위 클러스터 부트스트랩 B=60 (행 단위 아님 — 동일 선수 다중 시즌의
      군집 구조 존중), GroupKFold 5겹 밖 표본으로 커버리지 실측 검증
핵심 발견:
- 순수 부트스트랩 구간: 명목 80% → 실측 커버리지 29.3% (심각한 과소평가)
  원인: 부트스트랩은 모델·표본 불확실성만 포착, 미래 성과 자체의 변동(잔차) 누락
- 보정: sigma = sqrt(sigma_model^2 + sigma_residual^2), 잔차 sigma=7.49
  → 실측 커버리지 80%구간 82.0%, 50%구간 52.1% (명목치와 일치, 보정 성공)
- 정직한 구간 폭: 80% 구간 평균 ±10점 — 2~3년 예측의 본질적 난이도 반영
데모: 2020 U21 예측 상위 = Mbappé, Vinicius, Foden, Nkunku, Havertz, Haaland...
      → 실제 황금세대와 일치. Sancho 하락(-3.7)도 실제 궤적과 부합
주의: 데모는 훈련표본 포함(in-sample)이라 구간 적중 99.7%는 과대 — 공정한
      수치는 폴드 밖 82.0%임을 명시
