선수 잠재력 분석

FBref 선수 ID를 입력해 실시간 분석을 실행하세요. (예: 82ec26c1)

시연 팁

  • 첫 분석은 실시간 계산(몇 초), 같은 선수 재분석은 캐시(0.x초)로 응답됩니다.
  • 결과 화면에서 리포트(HTML)와 이미지(레이더/막대)를 바로 볼 수 있습니다.
  • 역할·스타일·강점/약점·그룹 점수 등은 기존 코드 계산 결과입니다.

예시 흐름

  1. 선수 ID 입력 → 분석 시작
  2. 실시간 분석 배지 확인(소요시간)
  3. 다시 같은 선수로 실행 → 캐시 응답 배지 확인
  4. 리포트 열기 버튼으로 상세 HTML 보기
